提瓦特阶级论：七国打工人血泪史

发表于2026-03-02|更新于2026-03-02|漫谈
|总字数:2.6k|阅读时长:7分钟|浏览量:

前言：我为什么要写这种东西

因为凌晨五点睡不着，打开原神看到派蒙说”好想吃提瓦特煎蛋”的时候突然想到一个问题：

派蒙吃的这个蛋是谁下的？鸡是谁养的？饲料是谁种的？旅行者一路打打杀杀拯救世界，有没有付过一次饲料钱？

然后我就开始想提瓦特的阶级问题，一想就收不住了。

蒙德：表面自由实际躺平的后封建社会

蒙德号称”自由之城”，风神巴巴托斯两千年前赶走了暴君之后就跑去睡觉了。听着很浪漫对吧？翻译成人话就是：CEO创业成功后直接跑路了，留下一帮中层管理（西风骑士团）自己搞。

你看蒙德的权力结构：

  • 西风骑士团 = 事业编公务员，有编制有工资有社会地位
  • 冒险家协会 = 外包零工，干最危险的活拿最少的钱
  • 迪卢克 = 本地首富，名义上不参与政治但实际上半个蒙德都是他家产业，晚上还搞暗夜英雄那套
  • 温迪 = 创始人，现在是蒙德最穷的人，天天蹭酒喝

蒙德的”自由”说白了是有钱人的自由。迪卢克家的晨曦酒庄横跨半个蒙德城外，雇了一大帮人种葡萄酿酒。他当暗夜英雄是出于正义感没错，但哥们你有这个闲是因为你不用打工啊。

而普通蒙德市民呢？去看看NPC对话就知道了——花匠担心花卖不出去，铁匠抱怨生意不好，猫尾酒馆的迪奥娜天天上班累得要死但不能辞职因为她是唯一的调酒师。

风神给了蒙德自由，但没给蒙德社保。

璃月：资本主义黄金时代

璃月是提瓦特最好懂的国家，因为它就是赤裸裸的商业社会。

璃月七星 = 董事会。凝光 = 董事长兼CEO，住在天上的群玉阁里。字面意义的”住在天上”。普通璃月人抬头就能看到凝光的豪宅飘在云里，每天提醒你：阶级差距真的可以物理性地悬浮在你头顶。

（后来群玉阁被打下来了，凝光说”没关系我再造一个”。没关系。再造一个。有钱真好。）

璃月港的底层什么样？你去轻策庄看看那些NPC就知道了——年轻人全跑去璃月港打工了，村里只剩老人。莲花池的渔民天天出海冒生命危险，赚的钱可能还不够在璃月港吃一顿天理珍馐。

最离谱的是愚人众在璃月的银行渗透。至冬国的北国银行开在璃月港正中央，搞金融操控。翻译一下：外资银行进驻本地市场挤压民族产业。这不就是19世纪的租界吗米浴。

岩神摩拉克斯最后选择”退休”——你全提瓦特的货币都是用你的名字命名的，你退个什么休？比尔·盖茨说要退休做慈善，至少他的钱还叫美元不叫盖茨。

稻妻：封建军国主义的活教材

稻妻是提瓦特里阶级矛盾最炸裂的国家。

雷神将军搞了个”眼狩令”——把所有人的神之眼收走。翻译一下：国家没收生产工具。但问题是，你看九条裟罗、神里绫华这些贵族阶层，该有眼的还是有眼。眼狩令本质上是对底层群众的单方面缴械。

三奉行制度更离谱了：

  • 天领奉行 = 祭祀贵族（精神统治）
  • 勘定奉行 = 经济贵族（钱袋子）
  • 社奉行 = 文化贵族（舆论控制）

三个贵族家族垄断了军事、经济、文化三大权力。然后愚人众还渗透进去搞掉了不合作的，换上了自己人。底层反抗军呢？被愚人众当枪使了，打完仗才发现自己的”革命领袖”是外国势力的棋子。

绯木村灭村——整个村子的人被冲突波及全灭了，但是雷神将军还在一心同体那里吃甜点。

海祇岛的反抗军本质上就是一群被压迫到受不了的渔民。心海看着像个温柔的军师，但你想想，让一群渔民拿着破武器去对抗有神之眼的幕府军，这中间的战力差距跟让外卖骑手去跟特种兵对线差不多。

须弥：知识就是权力（字面意义）

须弥玩了一手更高级的。

教令院垄断了知识。不是那种”好大学才有好资源”的间接垄断，是”知识本身由神的意识驱动，教令院控制神的意识”的直接垄断。

底层须弥人能接触到什么？虚空终端里的公开信息。教令院的学者能接触到什么？直接跟世界树对接的知识库。这个信息差比ChatGPT免费版和企业版的差距还大。

而且教令院还搞了个”轮回”实验——让一帮人在虚幻的梦境里循环，把他们的意识抽取出来喂给人造神。一群学者用活人做电池，这不就是提瓦特版的Matrix吗？尼奥在哪里？

赛诺作为大风纪官负责”执法”，但他自己也承认教令院的规矩很多时候就是扯淡。一个执法者亲口说法律有问题，但还是得执行——这个设定写实到让人不适。

枫丹：法治社会的精致陷阱

枫丹是最像现代社会的国家，所以阶级矛盾也最隐蔽。

芙宁娜在审判庭上审了几百年的案子，看着风光无限。但实际上呢？她自己就是最大的受害者——被迫扮演水神，一个人承受整个国家的重量，不能告诉任何人真相。翻译成职场语言：一个实习生被迫冒充总监撑场子，撑了几百年不能辞职不能请假不能崩溃。

枫丹表面上法治完善、媒体自由、审判公正。但你看壁炉之家——“仆人”开的孤儿院，表面收养孤儿，实际培养特工。林尼和琳妮特从小在里面长大，被训练成魔术师兼间谍。提瓦特版的”资本家开福利院当劳动力蓄水池”。

而且枫丹还有个特别真实的设定：出版业非常发达，各种报纸和小说满天飞。但出版商蒙吕松雇愚人众当保镖来销毁不利证据——媒体自由只在不触碰利益的时候存在。

愚人众：提瓦特最大的血汗工厂

最后说说愚人众这个组织本身，因为它的内部阶级结构堪称提瓦特的缩影。

顶层：冰之女皇，一个人的意志驱动整个国家。十一执行官，被赐予”神灵般的权柄”，各个都是人形核弹级别的战力。

底层：先遣队。名字叫”先遣”，意思是”生死不问，你先上”。单兵实力薄弱，装备落后，被派到异国他乡执行任务。说白了就是炮灰。

而且愚人众底层士兵的日常特别有打工人那味：

  • 有个哥们被外派到枫丹摸鱼，结果来了个工作狂副队长，摸不了了，于是伪造公文跑去水文监测站躲着——本质上就是上班划水被抓后跑去仓库躲起来假装盘库存
  • 有个冰铳重卫士出现在轻策庄，甘雨二话不说把人打了一顿，结果审完发现人家只是来摘树莓的——异地出差连摘个果子都要被当间谍打
  • “公子”跟先遣队新兵对练，结果新兵个个开盾回血，把”公子”暴打——新员工入职培训把CEO打了，离谱但真实
  • 玩家社区叫他们”驴人众”——因为NPC小女孩口齿不清把名字念错了。连组织名都被叫错，底层打工人的尊严约等于零

愚人众执行官”富人”潘塔罗涅出身贫寒，靠自己爬到了掌管整个至冬国经济的位置。标准的”穷人家的孩子当了资本家”叙事——但他现在对底层员工好吗？看看先遣队的待遇就知道了。阶级跃迁的终点是忘记自己的出身，古今中外概莫能外。

旅行者：提瓦特唯一的真·无产阶级

最后说一个大家可能没想过的：旅行者（也就是你操作的那个空/荧）是全提瓦特最纯粹的无产阶级。

没有生产资料：所有武器都是捡的或者别人给的。
没有固定住所：睡帐篷，借住在别人家。
没有组织归属：冒险家协会？那是零工平台不是单位。
没有社会保障：受伤了吃个甜甜花就完事，没有医保。

旅行者唯一的”生产工具”是自己的身体和战斗能力。他/她靠出卖劳动力（帮人打怪、跑腿、送信）换取报酬（摩拉和好感度）。每到一个新国家都要从零开始建立关系网，帮当地神解决问题然后继续流浪。

而派蒙？派蒙是旅行者的精神抚慰品——一个不生产任何价值但持续消耗食物的吉祥物。旅行者连这点精神享受的钱都是自己打怪赚的。

更惨的是：旅行者帮七个国家的神解决了存亡级别的危机，但到现在没有一个国家给过他/她一套房。连个编制都没有。纳西妲给了一朵花，雷神给了一把刀，芙宁娜给了……一段回忆？

拯救世界七次，依然买不起璃月港的房子。

这不就是当代打工人的终极写照吗。

（如果冰之女皇最后真的搞了个”提瓦特苏维埃”我第一个加入愚人众，至少有编制。）
