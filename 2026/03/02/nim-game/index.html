Nim游戏：异或的胜利，数学的暴力

发表于2026-03-02|更新于2026-03-02|算法
|总字数:2k|阅读时长:7分钟|浏览量:

如果御坂美琴和十六夜咲夜玩取石子

想象一下这个场景：

三堆硬币，分别有3、5、7枚。御坂美琴先手，十六夜咲夜后手。每次从任意一堆取任意多个（至少1个），拿走最后一枚的人赢。

美琴是Level 5的超能力者，但超电磁炮在这里没用。咲夜能停止时间，但停了时间也算不出最优策略（除非她停下来偷偷查手机）。

这个游戏纯粹是数学。谁先掌握了1901年Bouton定理，谁就永远不会输。

没有例外。

Bouton定理（1901）

Charles L. Bouton于1901年在 Annals of Mathematics 上发表了论文”Nim, A Game with a Complete Mathematical Theory”，给出了Nim游戏的完整解：

对于局面 $(a_1, a_2, \ldots, a_n)$，定义尼姆和（Nim-sum）为各堆数量的按位异或：

$$S = a_1 \oplus a_2 \oplus \cdots \oplus a_n$$

当 $S = 0$ 时，当前玩家必败（P-position）。当 $S \neq 0$ 时，当前玩家必胜（N-position）。

翻译成人话：把所有堆的数量异或起来，等于0你就寄了。

洛谷 P2197，直接交就过。这可能是你见过的最短的AC代码之一。

证明：为什么是异或

不是”因为异或好用所以用异或”，是”博弈的数学结构天然就是异或”。

定义两种局面：

  • P-position（Previous player wins）：当前玩家必败
  • N-position（Next player wins）：当前玩家必胜

任何公平博弈的局面满足以下三条性质（Zermelo, 1913）：

(i) 终止局面是P-position
(ii) 从N-position出发，至少存在一步走到P-position
(iii) 从P-position出发，任何一步都走到N-position

现在验证 $S = a_1 \oplus a_2 \oplus \cdots \oplus a_n$ 确实把P/N分对了：

性质(i)：终止局面 $(0, 0, \ldots, 0)$ 的异或值为0。哈基米都能验证这个。

性质(ii)：设 $S \neq 0$，取 $S$ 的最高位为第 $k$ 位。则必存在某个 $a_i$ 的第 $k$ 位为1（否则异或出来第 $k$ 位不可能是1）。令 $a_i’ = a_i \oplus S$，则 $a_i’ < a_i$（因为第 $k$ 位从1变成了0，更高位不变）。将第 $i$ 堆从 $a_i$ 改为 $a_i’$ 是合法操作，操作后新的异或值为：

$$a_1 \oplus \cdots \oplus a_i’ \oplus \cdots \oplus a_n = S \oplus a_i \oplus a_i’ = S \oplus a_i \oplus (a_i \oplus S) = 0$$

性质(iii)：设 $S = 0$，假设对某堆从 $a_i$ 改为 $a_i’$（$a_i’ < a_i$，即 $a_i’ \neq a_i$），则新异或值为 $0 \oplus a_i \oplus a_i’ = a_i \oplus a_i’ \neq 0$。

三条性质全部满足。$\blacksquare$

这个证明的美感在于：你不需要构造什么复杂的策略树，整个博弈的必胜必败完全被一个代数不变量刻画了。数学的暴力美学。

回到美琴vs咲夜

三堆硬币：3, 5, 7。

先手美琴必胜。策略：$S = 1$，找一堆异或完能变小的。$3 \oplus 1 = 2 < 3$，从第一堆拿1枚（3→2）。

局面变成 (2, 5, 7)：$010 \oplus 101 \oplus 111 = 000$。咲夜陷入P-position。

从这里开始，不管咲夜怎么操作，美琴都能把异或值重新拉回0。咲夜的时间停止能力在数学面前毫无用处，除非她作弊偷偷放回去几个——但那违反规则。

芙宁娜在旁边看审判：”本庭宣判，先手方御坂美琴胜诉。证据为Bouton定理（1901）。”

Sprague-Grundy定理：博弈论的统一场论

Nim只是开胃菜。Sprague（1935）和Grundy（1939）各自独立证明了一个更强的定理：

任何有限无环的公平组合游戏（Impartial Combinatorial Game）都等价于某个单堆Nim游戏。

这个等价通过SG函数（Sprague-Grundy函数）建立。

定义 $\text{mex}(S)$（minimal excludant）为集合 $S$ 中最小的不在 $S$ 中的非负整数：

$$\text{mex}({0, 1, 3}) = 2, \quad \text{mex}({1, 2}) = 0, \quad \text{mex}(\varnothing) = 0$$

（mex本身就挺哈基米的——找最小的不在里面的。就像你去食堂发现1号窗口2号窗口都有人排，你去3号，3号也有人，但4号没有——你的mex就是4号窗口。）

对于博弈图上的每个状态 $x$，递归定义：

$$\text{SG}(x) = \text{mex}\left({\text{SG}(y) \mid y \in \text{next}(x)}\right)$$

终止状态没有后继，$\text{SG}(\text{终止}) = \text{mex}(\varnothing) = 0$。

核心定理：如果游戏由 $k$ 个独立子游戏组成，则

$$\text{SG}(\text{总游戏}) = \text{SG}(G_1) \oplus \text{SG}(G_2) \oplus \cdots \oplus \text{SG}(G_k)$$

当且仅当总SG值为0时当前玩家必败。

标准Nim是这个定理的特例：单堆 $n$ 个石子的SG值恰好就是 $n$。

变种1：每次最多取k个（Bounded Nim）

一堆 $n$ 个，每次取 $[1, k]$ 个。SG值打表：

规律：$\text{SG}(n) = n \bmod (k+1)$

多堆时异或各堆的SG值。类似于把标准Nim的堆大小全部对 $k+1$ 取模。

如果咲夜限制美琴每次最多拿3个，美琴的优势就没那么大了——但只要她会对4取模，还是能赢。

变种2：阶梯Nim（Staircase Nim）

$n$ 级阶梯，每级有若干石子。每次将某一级上任意多石子移到下一级，第0级（地面）上的石子移出游戏。

结论：只对奇数级做标准Nim。

直觉：对手在偶数级动了什么，你立刻把那些石子往下推一级到奇数级”还原”。偶数级的操作等于废棋。

数学上这可以用SG函数+博弈等价性严格证明，但直觉已经足够你AC了。

变种3：反Nim（Misère Nim）

规则改成”拿到最后一颗的人输”。

结论几乎不变，只有一个特殊情况需要处理：

  • 所有堆的大小都 $\leq 1$ → 堆数为奇数时先手输
  • 存在大于1的堆 → 跟正常Nim一样看异或值

为什么会这样？因为”存在大堆”时你可以通过精确操控把局面导向”每堆最多1个且堆数对对手不利”的状态。Misère的特判本质上是endgame的细微差异。

SG值出现周期性在绝大多数竞赛题中都成立（Sprague-Grundy周期性猜想，1956年Guy和Smith做了大量数值验证）。找到周期后O(1)回答每个query。

做题checklist

碰到博弈题：

  1. 是标准Nim？→ 异或
  2. 能转化成Nim？→ 算SG值
  3. 多个独立子游戏？→ SG异或
  4. SG值算不出闭式？→ 打表找周期
  5. 打表也没规律？→ 你可能读错题了

第5条不是开玩笑。如果奶龙来打ACM，它大概率在第5条卡住。
