Nim游戏：我靠异或赢了队友一顿烧烤

发表于2026-03-02|更新于2026-03-02|算法
|总字数:2.2k|阅读时长:7分钟|浏览量:

起因是一顿烧烤

ACM训练的时候我们队有个传统：吃饭的时候用筷子玩取石子。三堆花生米摆桌上，两个人轮流从任意一堆里拿，每次至少拿一颗最多拿完整堆，谁拿走最后一颗谁赢，输的人请客。

一开始我跟队友都是凭感觉拿，输赢全靠运气。直到有一天我在OI Wiki上看到了Nim游戏的必胜策略，从此队友再也没赢过我，白嫖了一个学期的烧烤。

队友：”你是不是开了？”

我：”我开的是数学。”

规则很简单

地上有 n 堆石子，数量分别是 a1, a2, …, an。两个人轮流操作，每次选一堆，从里面拿走至少1个至多全部。谁拿走最后一个石子谁赢。

就这么简单。但”谁必胜”这个问题困扰了数学家几百年，直到1901年哈佛的Charles Bouton一巴掌拍出了答案。

结论：异或为0就输了

把所有堆的数量异或起来：

如果 nim_value == 0，当前这个人必输（后手必胜）。

如果 nim_value != 0，当前这个人必赢（先手必胜）。

为什么是异或？

这个部分你可以选择”信了就完事”然后跳到下一节，也可以跟我一起想明白为什么。

先定义两种局面：

  • P局面（Previous，上一个人赢了）= 当前玩家必输
  • N局面（Next，下一个人赢了）= 当前玩家必赢

三条公理：

  1. 所有终止局面（没石子可拿了）是P局面
  2. 能走到P局面的局面是N局面（你能把对手逼到必输，你就赢了）
  3. 只能走到N局面的局面是P局面（怎么走都是把对手送到必赢，你就输了）

现在考虑异或值：

终止局面所有堆都是0，异或值是0 → P局面。符合规则1。

异或值不为0的时候，一定存在某种操作让异或值变成0。怎么操作？找到异或值最高位的那个1，一定有某堆在这一位上也是1（不然异或出来这位不可能是1）。把这堆的数量改成”这堆原来的值 XOR 整体异或值”，新值一定比原值小（因为最高位从1变成了0），所以这是一个合法操作。操作完之后整体异或值变成0。符合规则2。

异或值为0的时候，任何操作都会让异或值变成非0（你改了一堆的值，异或值不可能还是0，除非你没改，但你至少要拿一个）。符合规则3。

三条规则都满足。证毕。

你看，不需要什么高深数学，就是把”什么时候赢什么时候输”的条件一条一条验证过去。

一个堆的时候更直观

一堆石子，n个。你先手。

n = 0：没得拿，你输了。
n >= 1：你全拿走，你赢了。

所以一堆的时候先手永远赢（n > 0时）。异或值就是n本身，确实非0。没毛病。

两堆的时候开始有意思

两堆：a和b。

如果 a == b，先手必输。因为不管你在哪堆拿多少，对手在另一堆也拿同样多就行了。你们一直”镜像”操作，最后一个石子一定是对手拿的。

如果 a != b，先手必赢。先从多的那堆拿，拿到跟少的那堆一样多，然后进入 a == b 的局面，对手就输了。

a == b 的时候 a ^ b == 0，a != b 的时候 a ^ b != 0。跟结论完全吻合。

回到烧烤现场

知道了这个之后，花生米博弈就变成了纯计算题。

比如三堆花生米：3, 5, 7。

我要做的就是找一堆改掉让异或值变0。当前异或值是1，我可以把3变成 3^1=2，也就是从第一堆拿走1颗。局面变成2, 5, 7，异或值 010^101^111 = 000。对手怎么拿都是非0局面，我再异或回0。

队友：”你拿花生米之前为什么总要愣一下？”

我：在心里算异或。

Sprague-Grundy定理：万能博弈分析框架

Nim游戏好懂，但竞赛里的博弈题通常不是标准Nim。比如”每次最多拿k个”、”石子分成两堆”、”在图上走棋子”之类的变种。

这时候就需要Sprague-Grundy定理了。核心思想：任何公平组合游戏（两个人交替操作、信息完全公开、不能操作的人输）都等价于Nim游戏。

具体做法是给每个局面算一个SG值：

mex是”最小排除值”——一个集合里最小的不在集合中的非负整数。比如 mex({0,1,3}) = 2，mex({1,2}) = 0，mex({}) = 0。

如果游戏由多个独立子游戏组成，总SG值就是各子游戏SG值的异或：

SG值为0就输，非0就赢。跟Nim一模一样。

例子：每次最多拿k个

一堆n个石子，每次最多拿k个。这不是标准Nim了，但用SG函数秒解：

规律：SG(n) = n % (k+1)。

多堆的时候异或各堆的 SG 值就行。一堆 n 个石子每次最多拿 k 个，等价于一堆 n%(k+1) 个石子的标准 Nim。

例子：阶梯Nim

n级阶梯，每级上有若干石子，每次可以把某一级上任意多颗石子移到下一级。地面是第0级，移到地面上的石子移出游戏。不能操作的人输。

结论：只看奇数级的石子，对奇数级做标准Nim。

为什么偶数级不用管？因为对手在偶数级的任何操作（把偶数级石子移到奇数级），你都可以立刻把那些石子再移到下一个偶数级去”还原”，相当于对手白操作了一轮。

这种”某些石子是废棋”的洞察在竞赛里非常常见。

反Nim：最后拿的人输

如果规则变成”拿到最后一颗石子的人输”（Misère Nim），结论微调一下：

  • 所有堆都是0或1 → 堆数为奇数时后手赢（异或值为1但先手输）
  • 存在大于1的堆 → 跟正常Nim一样看异或值

打出来的SG值经常会出现周期性规律（这叫Sprague-Grundy周期猜想，虽然没被完全证明但竞赛里基本都成立），找到周期就能O(1)回答。

做题checklist

碰到博弈题的时候按这个顺序想：

  1. 是不是标准Nim？→ 异或判
  2. 能不能转化成标准Nim？→ 找等价关系
  3. 能拆成独立子游戏吗？→ 各自算SG值然后异或
  4. 以上都不行？→ 打表找规律
  5. 打表也看不出规律？→ 考虑是不是读错题了

最后一条不是开玩笑。ICPC里我见过不止一次有队伍博弈题WA了十几发最后发现规则理解错了。
